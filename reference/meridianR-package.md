# meridianR: R Interface to Google Meridian (Marketing Mix Modeling)

An R interface to Google's 'Meridian' Bayesian Marketing Mix Modeling
(MMM) library. Wraps the 'Meridian' 'Python' package via 'reticulate',
providing faithful bindings to its data, model, and analysis objects
together with an idiomatic R layer for loading data, configuring priors
and model specifications, fitting models, and analyzing results.
