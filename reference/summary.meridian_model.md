# Summarize a fitted Meridian model

Prints a compact overview of the model's data dimensions and sampling
status, and invisibly returns a list with that information.

## Usage

``` r
# S3 method for class 'meridian_model'
summary(object, ...)
```

## Arguments

- object:

  A `meridian_model`.

- ...:

  Unused.

## Value

A list with dimensions and sampling status, invisibly.
